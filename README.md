# Smart Exam Seat Allocation System

## Overview

The Smart Exam Seat Allocation System is a web-based application developed to automate the process of allocating students to examination halls. The system allows administrators to upload student details, configure examination halls, generate seating arrangements, and search for student seat information using their USN.

## Features

* Upload student details using CSV files
* Configure multiple examination halls and capacities
* Automatic seat allocation based on hall capacities
* Search student seat details using USN
* Interactive dashboard with modern UI
* Responsive and user-friendly interface
* Local storage support for hall configuration

## Working Procedure

1. Upload the CSV file containing student details.
2. Configure the number of halls and their capacities.
3. Save the hall configuration.
4. Generate seat allocation.
5. View allocated halls and capacities.
6. Search student details using USN.
